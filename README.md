## Steps to start server

#### Clone repository

```
git clone https://github.com/IshaanOhri/Spreen
```

#### Install packages

```
npm ci
```

#### Development environment:

- Create `dev.env` file and add environment variables according to `example.env`:

```
touch ./src/config/dev.env
```

- Start server:

```
npm run dev
```
