FEATURES


Easy to Use

Just follow the guidelines on the GitHub repository as to how to start the Spreen server and then follow the on screen instructions.

Fast

As Spreen shares content over a local network, it provides the best user experience with minimum latency and maximum quality.

No Cables. No Internet

The two devices are not required to be connected via cable or over the internet. They just need to be connected to the same local network.

Restrict Sharing

Share your entire screen or just a single window, the choice is totally yours.

Any Screen

Use any additional device with a web browser as a second screen.

Free

Spreen is an open source project and is therefore completely FREE.
